# CookVibes

**CookVibes** és una aplicació web interactiva dissenyada per fer que la planificació dels àpats i el procés de cuina esdevinguin una experiència organitzada, senzilla i lliure d'estrès.

🔗 **Enllaç de la web:** [https://lalangosta.github.io/CookVibes/](https://lalangosta.github.io/CookVibes/)

## Tecnologies utilitzades

Aquest projecte (només frontend) s'ha desenvolupat utilitzant les tecnologies següents:

* **Vue:** Framework reactiu per a la creació d'interfícies d'usuari dinàmiques i components modulars.
* **Vite:** Eina de construcció ultra ràpida per a entorns frontend.
* **JavaScript:** Lògica del costat del client.
* **HTML i CSS:** Estructuració semàntica i disseny d'interfície dinàmic, amb carrusels infinits i disseny responsive.
